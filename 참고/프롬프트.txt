한국거래소 유가증권시장 업무규정 시행세칙 PDF를 엑셀로 변환하는 파이썬 프로그램을 만들어야 함.

파싱 조건:

1. 기본 구조
   - 조(條) > 항(項) > 호(號)까지만 구분. 목(目), 세목(細目)은 본문에 포함.
   - "제1조(목적)" 형식에서는 "제1조"를 [조], 괄호 안 전체를 [조명]으로 사용.
   - "제6조의4(지수차익거래의 정의 및 형태<개정 2021. 3. 12.>)"처럼 '의4'나 "<개정 …>"이 붙어 있어도 괄호 안 전체를 [조명]으로 사용.
   - "제30조의2", "제31조의14"처럼 '의'가 붙은 조는 [조]에 원문 그대로 넣기(예: "제30조의2").

2. 삭제된 조문
   - "제5조 삭제 <2019. 7. 11.>"처럼 조 전체가 삭제라는 내용만 있는 경우:
     - [조] = "제5조"
     - [조명] = "삭제"
     - [항] = '0', [호] = '0'
     - [내용] = "제5조 삭제 <2019. 7. 11.>" 전체 문장

3. 항(項) 처리
   - ①, ②, ③ … 이 명시된 경우:
     - 각 항을 별도 행으로 저장.
     - [내용]에는 조 제목("제○조(조명)")을 빼고, 순수 조문 내용만 넣기.
     - [참조번호]는 "제○조제①항" 형식으로 저장.
   - ①이 없는 조(本文만 있는 조):
     - 조 전체 본문을 하나의 행으로 저장.
     - [참조번호] = "제○조"
     - [항] = '0', [호] = '0'
     - [내용]에는 "제○조(조명)"을 제거한 나머지 본문만 넣기.

4. 호(號) 처리
   - 항 아래에 1., 2., 3. … 형식의 호가 있는 경우:
     - (1) 먼저 해당 항 전체를 하나의 행으로 저장
         · [참조번호] = "제○조제①항"
         · [항] = '①', [호] = '0'
     - (2) 이어서 각 호를 별도 행으로 저장
         · [참조번호] = "제○조제①항제1호" 식으로 작성
         · [항] = '①', [호] = '1', '2' …
   - 항 없이 바로 1., 2. … 호가 나오는 경우:
     - (1) 조 본문을 먼저 하나의 행으로 저장
         · [참조번호] = "제○조"
         · [항] = '0', [호] = '0'
     - (2) 각 호를 별도 행으로 저장
         · [참조번호] = "제○조제1호" 식으로 작성
         · [항] = '0', [호] = '1', '2' …

5. 각주/부기 처리
   - 조 제목 옆이나 조문 끝에 있는
     · "[전문개정 2021. 3. 12.]", "[본조신설 2018. 12. 6.]", "[개정 2019. 7. 11.]" 등은
       → 해당 조 또는 항의 마지막 [내용]에 그대로 포함.
   - 별도의 행으로 빼지 말고, 관련된 조문 셀 안에 같이 둔다.

6. 구분자 처리
   - 모든 항 번호 기호(①, ②, ③ …)와 호 번호(1., 2., 3. …)는 본문에서 제거하지 말고 그대로 포함해서 저장.
   - 괄호, 마침표 등 원문 기호도 그대로 둔다.

7. 컬럼 정의
   - 최종 표의 컬럼은 다음 여섯 개로 고정한다.
     1) 참조번호
     2) 조
     3) 조명
     4) 항
     5) 호
     6) 내용

8. 값 규칙
   - 호가 없으면 [호] = '0'
   - 항이 없으면 [항] = '0'
   - [항], [호]는 항상 문자열로 저장(숫자가 아니라 문자 '0', '1' …).

9. 출력 형식
   - 첫 줄에 컬럼명을 넣고, 이후 각 행에 한 조문씩 넣어 표 형태로 출력.
   - 엑셀에 바로 붙여넣을 수 있도록, 컬럼 사이를 탭(\t)으로 구분한 TSV 형식으로 출력해줘.

10. 통계 시트
   - 전체 파싱이 끝난 뒤, 항(項) 개수, 호(號) 개수, 총 항목 수를 집계해서
     · "구분", "개수" 두 컬럼으로 된 작은 표도 같이 만들어줘.
   - 이 표는 엑셀의 "통계_시행세칙" 시트에 넣을 것이다.
